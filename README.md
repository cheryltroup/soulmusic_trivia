#Soul Music Trivia

 Soul Music Trivia is a trivia game, with questions from the 60's, 70's and 80's.
 

To use the Soul Music Trivia skill, try saying...

Alexa, start Soul music.
Alexa, launch Soul music.
Alexa, begin Soul music.
